Welcome your home escape room experience!

This webapp allows you to create and run an Escape Room in your own home.

Start by clicking ... at the bottom of the screen and then the Edit button.

A game consists of stages, which has the screen text that is displayed, and an exit code that must be displayed to advance to the next stage. Each stage will also have one or more clues that help the players discover the exit code. These clues are intended to be hidden around the house.

Clues have the text and the location printed on them so that you know where to hide them. You can print a clue sheet by clicking Clue Sheet at the bottom of the screen.

Start the game by clicking on the Play button at the bottom of the screen. The game can be reset by clicking the Reset button at the top of the screen. The timer can be paused by clicking it.

An example game has been populated if you have not loaded one already. Games can be loaded and saved in the Edit screen using the Export and Import buttons.

This app is hosted on Github. Check out the project here.

Enjoy!
